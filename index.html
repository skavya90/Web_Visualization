    Summary: Latitude Vs X

    The purpose of this project to analyze the weather of around 500 cities across the world of varying distance from the equator and visualize the observations.To accomplish this, we utilized a Python library and OpenWeatherMap API to pull data into a readable format.

    After importing the data into a csv file format, we used matplotlib to plot scatter plots to observe relation between temperatures and latitudes.This site also provides source data for cities plotted and explains the data trends based on other factors like Humidity,cloudiness and Windspeed near equator regions.
